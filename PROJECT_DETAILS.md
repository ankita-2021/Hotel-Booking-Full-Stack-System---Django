# 🏨 Hotel Booking Full Stack System

A modern **Full Stack Hotel Booking Web Application** built using **React (Frontend)** and **Django REST Framework (Backend)**.
The system allows users to browse hotel rooms, view availability using a calendar, and securely book rooms online.

The platform ensures **real-time booking validation**, preventing double bookings for the same room and date.

---

## 🚀 Tech Stack

🖥 Frontend: React
⚙ Backend: Django + Django REST Framework
🗄 Database: Django Default Database (SQLite)
🔗 API Communication: REST APIs

---

## ✨ Key Features

🔐 **User Authentication**

* Users must **register and login** before booking a room.
* Prevents unauthorized bookings.

🏠 **Home Page**

* Displays the main landing page of the application.
* Users can explore available hotel rooms and navigate easily.

🛏 **Room Listing**

* Shows all available hotel rooms with details and pricing.

📅 **Calendar-Based Booking**

* Users can select booking dates using an **interactive calendar**.

🚫 **Double Booking Prevention**

* If a room is already booked for a specific date, another user **cannot book the same room for that date**.

🧾 **Room Details Page**

* Displays detailed information about the selected room.

📌 **Secure Booking Workflow**

* Booking requests are validated by the backend to ensure correct availability.
